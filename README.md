# MSMD-PNC-Dataset
## Data Description
This study introduces a multi-source meteorological dataset named MSMD-PNC (Multi-source Meteorological Datasets Based on Precipitation Nowcasting) to advance research on precipitation nowcasting. The dataset incorporates rainfall radar data, weather model data, and satellite data from northwestern France. The MeteoNet dataset is cleaned and standardized, with uniform formatting applied across all data sources and file types, including temporal resolution, spatial resolution, and file formats. The latitude and longitude coverage of the data are reorganized according to data quality.

The dataset is derived from MeteoNet (https://meteonet.umr-cnrm.fr/), developed and provided by Météo-France, the French national meteorological service. MeteoNet is a project of Météo-France that collects various meteorological data over France, including surface observations, precipitation radar data, weather model outputs, satellite data, land–sea masks, and terrain elevation data. Each available data type covers a three-year period from 2016 to 2018, spanning a geographic region of 550 km × 550 km along the Breton coast, covering northwestern and southeastern metropolitan France.
